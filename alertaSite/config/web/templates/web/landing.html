Что такое Alerta?

Создавайте проекты, добавляйте HTTP/ICMP/TCP мониторы, отслеживайте аптайм и реагируйте на инциденты из браузера или Android-приложения.

AI-ассистент помогает анализировать сбои, REST API интегрируется с вашими инструментами, а публичные status page держат клиентов в курсе.
